# AGENTS

This repository is a GitHub Pages site built with Jekyll.

## Project overview
- The site is published through GitHub Pages.
- Content and site structure are managed with Jekyll.
- Changes should preserve the expected Jekyll-compatible layout and front matter.

## Development notes
- Keep content and configuration compatible with Jekyll and GitHub Pages.
- If adding pages or posts, use Jekyll conventions such as front matter at the top of files.
- Prefer simple, static-site-friendly changes that work well in a GitHub Pages deployment.
